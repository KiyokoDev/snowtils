# Snowtils

> **Disclaimer:** This mod automates in-game actions which may be against Hypixel's rules. Use at your own risk. I am not responsible for any bans or penalties.

Auto-clicks bracketed dialogue options in Hypixel Skyblock chat so you don't have to mash through NPC conversations manually.

## Features

- Automatically clicks any bracketed `[text]` options in NPC chat messages
- Only clicks the first option in a message
- 350ms delay before clicking to let the dialogue settle
- Each command is only clicked once — no spam
- Won't click while any GUI screen is open
- Toggle on/off with **H** key (configurable in controls)

## Usage

1. Join Hypixel Skyblock
2. Talk to an NPC
3. The mod auto-clicks the first bracketed dialogue option for you
4. Press **H** to toggle on/off (status shown in chat)
